Lab 1: PowerBI

An introduction to a SaaS analytics platform

Objective

Get a high-level overview of analytic visualization software like PowerBI, Tableau and Qlik.

In this lab we'll be briefly going through PowerBI and how it is used.

Introduction

PowerBI, Tableau, Qlik are three of the most popular business intellgence tools out there.

  • Power BI
  • Tableau
  • Qlik

PowerBI users

  • Analysts: use Power BI to develop reports, dashboards, data models and study them to discover valuable insights in the data.
  • Business Users: end users who look at the reports shared to them on the PowerBI website / mobile app.
  • Developer: When a custom report or data manipulation is required.
  • IT administrator: When you host data and are concerned about scalability, security of data.

PowerBI is typically more for analysts and users.

An analogy: You use Excel, and it is useful for quick work.

But as a computer science student you can do way more than that with code.

Data connections in PowerBI

  • File: Excel, Text/CSV, XML, PDF, JSON, Sharepoint, etc.
  • Database: SQL DB, Access, Oracle DB, etc
  • Azure: Azure SQL, Azure Analysis Services, Azure Data Lake, Azure Cosmos DB, etc.
  • Online Services: Salesforce, Google Analytics, Facebook, GitHUb, etc.
  • Others: Python code, R code, D3.js code, etc.
